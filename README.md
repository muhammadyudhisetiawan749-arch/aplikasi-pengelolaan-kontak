## Deskripsi
Program ini merupakan aplikasi Java berbasis JFrame yang berfungsi untuk mengelola daftar kontak (nama, nomor telepon, dan kategori).  
Aplikasi ini mendukung fungsi CRUD (Create, Read, Update, Delete) menggunakan koneksi database MySQL.

---

## Struktur Proyek
- `nbproject/` : Folder konfigurasi NetBeans  
- `src/` : Folder kode sumber aplikasi (Java)  
- `build.xml` : File build otomatis dari NetBeans  
- `manifest.mf` : Informasi project dan main class  
- `README.md` : Deskripsi dan identitas pembuat
